# Wine Quality Prediction 🍷

This project explores predicting wine quality using machine learning techniques at two levels: basic and advanced.

The goal is to understand how different chemical properties of wine influence its quality and to build models that can predict it.

## Project Overview

- Dataset: Wine Quality dataset
- Task: Predict wine quality based on input features
- Approach: Data analysis, preprocessing, model training, and evaluation

## Files

- `Wine Quality_basic.ipynb` → Beginner-friendly implementation with simple preprocessing and models  
- `Wine Quality_advanced.ipynb` → Improved version with better techniques and deeper analysis  

## Learning Outcome

- Understanding dataset features and relationships  
- Building basic and advanced ML models  
- Comparing performance improvements
